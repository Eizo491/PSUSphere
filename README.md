# 🎓 PSUSphere: Student Organization Management System

**PSUSphere** is a web-based application developed for **Palawan State University** to manage student organizations, track memberships, and organize college-specific programs. This project was built as part of the Django web development curriculum.

## 📝 Description
The system provides a centralized platform for administrators to:
* Manage university **Colleges** and their respective **Bachelor Programs**.
* Maintain a database of **Students**.
* Oversee student **Organizations** (like ACS and SITE).
* Track **Organization Memberships** and join dates.

## 🚀 Key Features
* **Custom Admin Dashboard:** Refactored Django Admin with search and filtering capabilities.
* **Data Automation:** Integrated with `Faker` to generate large amounts of test data for students and organizations.
* **Relational Database:** A clean architecture linking students to their programs and organizations.

## 🛠️ Tech Stack
* **Framework:** Django (Python)
* **Database:** SQLite (Development)
* **Data Generation:** Faker Library
* **Version Control:** Git & GitHub
